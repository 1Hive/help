Aragon is software for creating and governing organizations such as clubs, companies, gaming guilds, cooperatives, nonprofits, open source projects, and any other type of organization you can imagine. Aragon currently supports creating organizations on the main Ethereum network (“mainnet”) and the Rinkeby test network (“testnet”).

You can find more information about the Aragon project, decentralized organizations, governance, and other related topics in the Aragon wiki.
